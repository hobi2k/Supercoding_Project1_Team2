- 자기 브랜치 설정
1. git 설치: https://git-scm.com/downloads/win
2. 설치한 폴더에서 터미널을 열고 다음을 입력하여 로컬에 설치: git clone https://github.com/hobi2k/Supercoding_Project1_Team2.git
3. 설치된 폴더에서 터미널을 열고 다음을 입력 git config --global --add safe.directory "(폴더 path)D:/프로그래밍/슈퍼코딩 DATA.AI/project1/Supercoding_Project1_Team2"
#해당 폴더는 안정한 폴더로 설정된다.
4. git checkout main를 입력하여 main 브랜치로 이동한다.
5. git checkout -b 새 브랜치 이름 을 입력하여 자기 브랜치를 생성하고 해당 브랜치로 이동한다.
#git branch -a로 로컬 저장소에 생성되어 있는 브랜치 확인
#git branch -r로 원격 저장소에 생성되어 있는 브랜치 확인
6. git branch --set-upstream-to origin/main로 원격 저장소에 자기 브랜치를 업로드한다.
#이후에는 git checkout 내 브랜치 이름 으로 자기 브랜치로 이동하면 된다.

- 작업물 업로드
#git status 현재 로컬 저장소 상태를 본다.
1. git add .   // 로컬저장소에 있는 모든 파일을 추가한다.
2. git commit -m "[hosung] README 수정"  방금 추가한 파일들 업로드를 준비한다. 누가 어떤 내용을 수정/추가해서 올리는지 작성한다.
3. git push origin 내 브랜치 이름 를 입력하여 원격 저장소에 업로드한다.
